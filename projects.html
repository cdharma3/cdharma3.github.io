Timings Comparison Application

Company: Teradyne

When: Summer 2021

The timings comparison application is an application created by a team of three interns working in an agile based project during the summer. The application is a GUI based application written in C#, it takes two log files stored in a CSV file and graphically displays the differences between the log files.

We scoped and planned the majority of the project ourselves, only receiving a minor set of requirements from our mentors. The entire project was done using the Agile methodology, our team was responsible with writing user stories and planning the two week sprints the project was broken up into.

My primary responsiblities included designing and coding the comparison backend that performed the comparisons between the log file entries, aiding in the integration of the GUI with said backend and integration testing.
